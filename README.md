# AI-Annotation-Portfolio
Street scene image data annotation

Overview

This project demonstrates my practical experience in image annotation for computer vision tasks using Label Studio.

Project Details

- Task Type: Object Detection
- Annotation Method: Bounding Boxes
- Tools Used: Label Studio

Labeled Objects

- Person
- Car
- Bicycle
- Traffic Light

Skills Demonstrated

- Data annotation and labeling
- Bounding box annotation
- Annotation quality control
- JSON data structure understanding
- Occlusion and visibility handling

Annotation Standards

- Minimum 50% visibility threshold
- One bounding box per object
- No duplicate annotations
- Consistent labeling

File Structure

- "data/": Sample images
- "annotations/": JSON annotation files
- "screenshots/": Annotation process evidence
- "docs/": Guidelines followed

Sample Output

Annotations are provided in JSON format, including bounding box coordinates and object labels.

Portfolio Note

This project reflects my ability to produce accurate and structured annotation data suitable for training AI models.
